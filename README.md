A simple yet effective Python application that reminds you to stay hydrated by sending periodic desktop notifications.

## ✨ Features

- 🕒 **Regular Reminders**: Sends a notification every hour to remind you to drink water.
- 🖥️ **Desktop Notifications**: Uses system-level notifications for maximum visibility.
- ⚙️ **Lightweight**: Minimal resource usage, runs smoothly in the background.

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed on your system. You also need the `plyer` library for notifications.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Drink-Water-Reminder.git
   cd Drink-Water-Reminder
   ```

2. **Install dependencies:**
   ```bash
   pip install plyer
   ```

## 🛠️ Usage

Simply run the script to start receiving reminders:

```bash
python main.py
```

The script will notify you immediately and then repeat every hour.

## 📦 Dependencies

- [plyer](https://pypi.org/project/plyer/) - A Python library to access features of your hardware/platforms.
